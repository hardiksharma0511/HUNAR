# 🎨 HUNAR

### Handmade Marketplace for Indian Artisans

> **Preserving Indian craftsmanship through technology.**

HUNAR is a full-stack marketplace where artisans can showcase and sell handmade products directly to customers. The platform focuses on traditional Indian art while providing a modern and seamless shopping experience.

---

## ✨ Highlights

- 👤 Buyer & Seller Authentication
- 🎨 Artisan Profiles
- 🛍️ Product Listings & Categories
- ❤️ Wishlist & Shopping Cart
- 📦 Order Management
- 🔍 Search & Filters
- ☁️ Cloudinary Image Uploads
- 📱 Fully Responsive Design
- 🔒 Secure JWT Authentication

---

## 🎯 Why HUNAR?

Unlike traditional e-commerce platforms, HUNAR is designed specifically for handmade products and independent artisans. It combines a modern shopping experience with a warm, culturally inspired interface that celebrates Indian craftsmanship.

---

## 📸 Features

### Buyers
- Browse handmade products
- Search & filter by category
- Wishlist & Cart
- Secure checkout
- Order history
- Artisan profiles

### Sellers
- Seller Dashboard
- Add, edit & delete products
- Upload product images
- Manage incoming orders
- Track sales

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/hardiksharma0511/HUNAR.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
