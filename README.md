# 🎨 HUNAR
### Handmade Marketplace for Indian Artisans

> **🚧 Work In Progress**
>
> HUNAR is currently under active development. New features, UI improvements, and backend enhancements are being added regularly.

---

## 🌟 About

HUNAR is a full-stack marketplace built to connect talented Indian artisans with buyers looking for authentic handmade products.

The platform allows artisans to showcase their work while providing customers with a seamless shopping experience.

---

## ✨ Features

- User Authentication (Buyer & Seller)
- Seller Dashboard
- Product Listings
- Product Categories
- Wishlist
- Shopping Cart
- Order Management
- Search & Filters
- Contact Seller
- Image Uploads
- Responsive Design
